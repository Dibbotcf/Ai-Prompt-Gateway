# AI Prompt Security Gateway

Professional security dashboard for analyzing and filtering AI prompts.

**Live Demo:** https://aipromptgateway.astrozupi.com

## Features
- Real-time threat detection with 44 detection rules
- 5 attack categories: Jailbreak, PII, Harmful Intent, System Override, Social Engineering
- AI model integration (Groq/Gemini/OpenAI)
- Test bench with activity sessions
- Professional enterprise security dashboard aesthetic

## Tech Stack
PHP 8  / MySQL / Vanilla JS / Chart.js 4 / IBM Plex Sans/Mono
